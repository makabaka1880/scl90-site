# SCL-90

A web-based Symptom Checklist-90 (SCL-90) self-assessment tool built with Vue 3 + TypeScript.

## Features

- **90-question psychological assessment** across 10 dimensions
- **Bilingual support** (English / Chinese)
- **Real-time scoring** with GSI calculation
- **Visual analysis** with dimension breakdowns

## Tech Stack

- Vue 3 + TypeScript
- Pinia (state management)
- vue-i18n (internationalization)
- Vite
- SCSS

## Development

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
```
